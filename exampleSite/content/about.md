## About This Theme

**Devtronic** is a minimal, developer-focused Hugo theme inspired by the digital aesthetics of the Matrix. It combines emerald phosphor glow with modern web design principles to create a unique, professional blogging experience.

### Design Philosophy

The theme is built on three core principles:

1. **Minimal & Clean**
   - Zero JavaScript core
   - Lightweight CSS
   - Fast page loads

2. **Developer-Focused**
   - Beautiful code highlighting
   - Monospace typography options
   - Terminal-inspired aesthetics

3. **Accessible & Professional**
   - WCAG AA compliant
   - Keyboard navigation
   - Semantic HTML5
   - SEO optimized

### Technical Highlights

- **CSS Grid & Flexbox** for modern, responsive layouts
- **Emerald accent palette** (#00FF99) throughout
- **Hugo asset pipeline** for optimized CSS delivery
- **Open Graph & Twitter Cards** for social sharing
- **RSS feeds** built-in
- **Chroma syntax highlighting** with custom emerald theme

### Perfect For

- Developer blogs
- Technical journals
- Digital portfolios
- Tech documentation sites
- Anyone who appreciates clean, minimal design
